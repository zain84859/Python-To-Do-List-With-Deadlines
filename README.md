# Python Task Manager

This is a GUI-based task management application developed using Python and Pygame.

## Features
- Add, edit, and delete tasks with deadlines
- Revision planning algorithm based on task difficulty
- Priority system based on deadline and difficulty
- Notification system using Plyer
- Data stored using SQL database
- Fully object-oriented and modular code

## What I learned
- Object-oriented programming (OOP)
- Working with databases (SQL)
- Designing algorithms
- Debugging and testing
